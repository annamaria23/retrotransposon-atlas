# Resources associated with TE annotations

- [annotated_TEs.fa](annotated_TEs.fa) contains the consensus sequence of each annotated TE
- [annotated_TEs.gtf](annotated_TEs.gtf) contains the GTF files with locations of ORFs wihin each TE
- [TE_POL_trees](./TE_POL_trees) contains MSAs and phylogenetic trees of all _POL_ ORF sequences per TE
- [TE_Locations](./TE_Locations) includes the genomic locations of _Gypsy_-family TEs, as obtained by RepeatMasker
